# 🔍 Dev Detective — GitHub User Analyzer

## 📌 Overview

Dev Detective is a web application that allows users to search GitHub profiles and compare two users using a battle mode.
The app fetches real-time data using the GitHub API and displays it in a clean, animated UI.

---

## 🚀 Features

### 🔹 Search Mode

* Search any GitHub username
* Displays profile details:

  * Avatar
  * Name
  * Bio
  * Followers
  * Score (followers + stars)

### 🔹 Battle Mode (Level 3)

* Compare two GitHub users
* Calculates score based on followers and stars
* Highlights winner and loser visually

---

## ⚡ Advanced Features

* Fetch API integration (GitHub API)
* Async/Await handling
* Error handling (User Not Found)
* Loading skeleton UI
* API fallback mechanism (handles rate limits)
