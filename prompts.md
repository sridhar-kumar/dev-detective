# 📌 Prompt Log — Week 3 Project (Dev Detective)
1. Create a web app using HTML, CSS, and JavaScript to search GitHub users using the GitHub API.
2. Add functionality to display user profile details such as avatar, name, bio, and followers.
3. Use fetch API with async/await and handle errors properly when user is not found.
4. Show error message if username is invalid or not found.
5. Create a battle mode where two GitHub users can be compared based on followers and repository stars.
6. Calculate a score using followers and stars, and highlight winner and loser visually.
7. Upgrade UI to a modern dark theme with glassmorphism and smooth animations.
8. Handle API rate limit issues and prevent app from breaking.
9. Ensure all Week 3 requirements are satisfied and the app is fully functional with a clean UI and smooth user experience.
